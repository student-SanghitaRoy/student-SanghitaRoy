## 🚀 Featured projects

### 🧠 NeuraScan AI — Brain Tumour Detection System
> An end-to-end deep learning project for multiclass brain tumour classification from MRI images, with Grad-CAM explainability and an interactive web deployment.

**Features**
- Multiclass brain MRI classification
- Ensemble learning across multiple CNN architectures
- Grad-CAM heatmaps for visual explainability
- Interactive web app deployed on Hugging Face Spaces
- Professional documentation

**Models used:** VGG16 · ResNet50 · EfficientNetB0

**Stack:** `Python` `TensorFlow` `Keras` `Flask` `Hugging Face Spaces`
